# Wireframe Notes

## Wireframe 1: Main Task Tree View

The main screen should show a sidebar, a central hierarchical task tree, and a selected task details panel.

## Wireframe 2: Task Details/Edit Panel

The details panel should show task title, parent task, root task, estimate, completion status, and scheduling actions.

## Wireframe 3: Analog Clock Time-Blocking View

The analog clock view should use a 12-hour clock face to visually place task blocks into a morning, afternoon, or evening planning period.

## Wireframe 4: Vertical Day View

The vertical day view should show a conventional top-to-bottom daily timeline with scheduled task blocks.

## Wireframe 5: Scheduling Interaction

The user should be able to select a task, choose a start/end time, choose whether completing the time block completes the task, and then see that block in both scheduling views.
